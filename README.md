In this work, I have perform Time Series Analysis on Global Active Power from Household energy consumption (https://archive.ics.uci.edu/ml/datasets/individual+household+electric+power+consumption)
The process involved 
* Data Preprocessing : invalid handling, Missing Value handling, etc.
* Visualization: Time series plot, seasonal plot, ACF plot, distribution plot
* Statistical test, ADF test, Ljung-Box, etc.
* Model specification: SARIMA
* Model Diagnostic: Observe Residuals
